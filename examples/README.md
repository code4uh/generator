# Beispiele

- `json/simple_layout.json`: Kleines, gut lesbares Beispiel mit zwei Devices, je einem Pin und zwei WireTiles auf unterschiedlichen Layern.
- `json/complex_layout.json`: Reichhaltigeres Beispiel mit drei Devices, unterschiedlichen Layer-Spannen, mehreren Pins mit Mehrfach-Ports (OR-Semantik) und mehreren WireTiles über mehrere Layer.
